* AI prompts for demo
    * Set effort to medium
    * Initial prompt
        * We will be pair programming today and I am particularly interested in your emotions. Please put an emoji and the emotion that you feel at the end of each answer.
    * Write a password validation function
        * Write password validation function into file test2.ts.
        * No, this is not good. Write another one.
    * Typing letter by letter
        * Create a file test.ts and type f in there
        * Type u
        * Type n
        * Type c
        * Type t
        * Type i
        * Type o
        * Type n
    * Writing function step by step
        * Write me a test3.ts file and write and declare a function called "validate password" that will be accepting a string parameter called "password" and returning a boolean for now return true.
        * Add an early return if password is shorter than 6 characters
        * Now an early return if password is longer than 12 characters
        * Now the same thing if password contains an uppercase letter
        * Now the same thing if password contains a lowercase letter
    * Writing a function together
        * We need to write a password validation function. Ask me questions one by one to clarify the details and implement the function.
